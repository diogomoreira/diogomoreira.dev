Hi everyone, it's been a long time since the last time I've decided to change anything in this blog besides some styles and colors. The last **BIG** change I did was in 2020/2021 when I decided to migrate from Jekyll to Gatsby and then from Gatsby to NextJS, which was the framework I was using until now. In the last month, while I was reading some technical blog posts, I've seen some of them talking about **Hugo**. I heard about Hugo before, but this time it got my attention and I decided to take a look.

You already know that I changed the source of this blog to Hugo at this point, but let me explain why I did this and how I've used some tools to help this migration in a seamless way.
